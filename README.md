# What is it about?

This is a test project where I play with different file types and data returned from the server.

This way you can find:
* simple cases like sending inlined/attached test, json etc. in [SimpleDataController](src/main/java/org/botnicholas/projects/demofiles/controllers/SimpleDataController.java)
* small files inlined/atached in [SmallFilesController](src/main/java/org/botnicholas/projects/demofiles/controllers/SmallFilesController.java)
* streaming big files and generated data in [BigAndGeneratedFilesController](src/main/java/org/botnicholas/projects/demofiles/controllers/BigAndGeneratedFilesController.java)

You can start with [this](http://localhost:8080/swagger-ui/index.html#/) page
